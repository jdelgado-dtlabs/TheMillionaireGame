## Step 1: Launch and Initial Setup (1 minute)

### Launch the Application
1. Double-click the desktop shortcut or Start Menu entry
2. Two windows open:
   - **Control Panel** - Your operator interface
   - **TV Screen** - The player/audience display

### First-Time Setup
On first launch, the application will:
- Initialize the database
- Create default configuration
- Load sample questions (50 questions included)

> üí° **Tip**: If using dual monitors, drag the TV Screen to your primary display (TV/projector) now.

---

## Step 2: Configure Display (30 seconds)

### Single Monitor
- Arrange windows: TV Screen maximized, Control Panel below or on second desktop

### Dual Monitor (Recommended)
1. **Primary Monitor** (TV/projector): TV Screen (full screen)
2. **Secondary Monitor**: Control Panel (operator view)

### Set Full Screen (Optional)
- Click "Full Screen" button in Control Panel
- TV Screen expands to cover entire display
- Press `ESC` or `F11` to exit full screen

---

## Step 3: Create a Game Profile (1 minute)

A game profile stores your game settings (money tree, lifelines, currency).

1. In Control Panel, click **"Game Profiles"** tab
2. Click **"New Profile"**
3. Enter profile details:
   - **Name**: "My First Game"
   - **Currency**: Select your preferred currency (e.g., USD, EUR)
   - **Money Tree**: Use default or customize prize amounts
   - **Lifelines**: Keep defaults (50:50, Phone, Audience, Switch)
   - **Safety Nets**: Milestone amounts (default: $1,000 and $32,000)
4. Click **"Save Profile"**

> üí° **Tip**: Use the default settings for your first game. You can customize later!

---

## Step 4: Load Questions (30 seconds)

The application includes 50 sample questions for testing.

1. In Control Panel, go to **"Questions"** tab
2. Verify "Default Question Set" is selected
3. Questions are organized by difficulty (5 levels, 15 questions total for main game)
4. Click **"View Questions"** to preview available questions

> üìù **Note**: You can add more questions later using the Question Editor or importing CSV files.

---

## Step 5: Start a New Game (30 seconds)

### Choose Game Mode

#### Option A: Main Game (Single Player)
Best for testing or when you have one contestant ready.

1. Click **"New Game"** button
2. Select your profile: "My First Game"
3. Click **"Start Main Game"**
4. Enter contestant name (optional)
5. Click **"Begin"**

#### Option B: Fastest Finger First (Multiple Players)
Best for authentic game show experience with multiple contestants.

1. Click **"New Game"** button
2. Select your profile: "My First Game"
3. Click **"Start FFF"** (Fastest Finger First)
4. Enter contestant names (2-8 players)
5. Click **"Begin FFF Round"**

---

## Step 6: Play the Game (2 minutes)

### Fastest Finger First (If Started)

1. **FFF Question Appears** on TV Screen
2. Contestants race to order items correctly
3. **Control Panel**: Enter times and answers for each contestant
4. Click **"Show Results"**
5. Winner becomes the main game contestant

### Main Game

#### Question Flow
1. **Question Appears**: TV Screen shows question and 4 answers
2. **Contestant Considers**: Thinking music plays automatically
3. **Host Reads Question**: Click "Read Question" or use hotkey (`Space`)
4. **Contestant Answers**: They verbally give their answer (A, B, C, or D)

#### Recording Answer
1. In Control Panel, select the contestant's chosen answer (A/B/C/D)
2. Click **"Final Answer"** (or press `Enter`)
3. Dramatic pause...
4. Correct/Wrong answer revealed automatically

#### Using Lifelines
Contestant can use lifelines before giving their final answer:

**50:50** (`F1` key):
- Click "50:50" button
- Two wrong answers are eliminated
- TV Screen updates showing only 2 answers

**Phone a Friend** (`F2` key):
- Click "Phone a Friend" button
- 30-second timer starts
- Pause game with spacebar when done

**Ask the Audience** (`F3` key):
- Click "Ask the Audience" button
- Audience votes (manually enter percentages or use web integration)
- Results shown as bar chart

**Switch Question** (`F4` key):
- Click "Switch Question"
- New question of same difficulty loaded
- One-time use only

#### Walking Away
If contestant wants to quit:
1. Click **"Walk Away"** button (or press `W`)
2. Contestant keeps current winnings
3. Game ends

#### Winning or Losing
- **Correct Answer**: Prize won, proceed to next question
- **Wrong Answer Below Safety Net**: Drop to safety net amount, game ends
- **Wrong Answer Above Safety Net**: Lose everything above safety net
- **Reach Top Prize**: Confetti animation, game won!

---

## Step 7: Review Results (30 seconds)

After game ends:

1. **Final Screen**: Shows final winnings and performance
2. **Game Console**: Review game log in Control Panel
3. **Telemetry**: Game automatically saved to database

To view statistics:
- Click **"Telemetry"** tab
- See all games played
- Export to Excel (XLSX) for detailed analysis

## Quick Tips for Your First Game

### Audio Controls
- **Master Volume**: Control Panel ‚Üí Audio section
- **Music**: Background tension music (auto-adjusts by level)
- **Effects**: Button sounds, correct/wrong answers
- **Mute Shortcut**: `M` key

### Display Issues
- **TV Screen Too Small**: Click maximize or use full-screen mode
- **Control Panel Obscured**: Move to second monitor or minimize during reveals

### Game Flow
- **Don't Rush**: Let sound effects and animations play
- **Use Hotkeys**: Faster than clicking buttons (see [Hotkey Reference](#hotkey-reference))
- **Pause Anytime**: Press `P` to pause game

### Common Mistakes
- ‚ùå Clicking "Final Answer" before contestant confirms
- ‚ùå Forgetting to enable full screen for TV Screen
- ‚ùå Not testing sound levels before live event
- ‚úÖ Run a complete practice game first!

## Troubleshooting Quick Fixes

### TV Screen Not Visible
- Check if behind Control Panel
- Press `F11` for full screen
- Move to correct monitor

### No Sound Playing
- Check Windows volume mixer
- Verify sound files in `lib\sounds\Default\`
- Test with volume slider in Control Panel

### Questions Not Loading
- Verify database connection
- Check Questions tab for errors
- Restart application

### Game Frozen/Crashed
- Application includes automatic crash monitoring
- Check `Logs\` folder for error reports
- See [Troubleshooting](Troubleshooting) for details
